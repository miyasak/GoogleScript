# プログラム概要

BigQueryのテーブルスキーマを元に、REQUIREDモードのフィールドを取得してソース元のスプレッドシートにあるフィールド名と一致した列に対してデータの入力規則と入力書式をセットします。<br>
フォーマットがあっていれば、複数の外部データシートに対して入力書式のセットが可能です。<br>

# 事前準備

A.BigQueryのデータベースに登録されている各テーブルスキーマの情報を以下のような形でスプレッドシートにまとめておきます。<br>
<img width="1358" alt="スクリーンショット 2022-01-16 22 30 10" src="https://user-images.githubusercontent.com/43813301/149662019-1e4ce732-0620-42bd-a0e3-ca11c78ba080.png">
尚、このスプレッドシートは以降、テーブルシートとします。<br>

B.実際にBigQueryへ読み込む外部データのスプレッドシートを項目毎に並べます。この時項目名はテーブルスキーマの説明と一致していることが必要なことに注意してください。<br>

尚、このスプレッドシートは以降、外部データシートとします。<br>

# 使い方

1.テーブルシートのGoogleAppScriptを開いて、main.jsとconfig.jsをコピーします。<br>
2.config.jsを開いて、テーブル定義の項目名を調整します。<br>
3.main.jsを開いて、セルの入力書式にセットするカスタム数式を調整します。 setting.ruleオブジェクトのformatが該当します。<br>
4.3と同様にデータの入力書式にセットするカスタム数式を調整します。setting.ruleオブジェクトのinputが該当します。<br>
5.関数Main()を実行します。<br>

# フォルダ構成

・README.md：このファイル<br>
・config.js：設定ファイル<br>
・main.js:メインロジック。外部データシートに対してセルの入力書式とデータ入力規則をセットする<br>

# 注意点
* 各シートのA1セルには、外部データシートのリンクを埋め込んでください!!<br>
* 各シートのヘッダはA列：フィールド名、B列：種類、C列：モード、D列：説明の順にセットしてください!!<br>
* 外部データしーとにセットする入力書式もしくはデータ入力規則のカスタム関数を変更する場合は、setting.ruleオブジェクトの値を書き換えてください!!<br>
* モード指定を変更する場合は、TARGET_MODEの変数名を書き換えてください。(現時点では複数のモードには未対応です)<br>
* 外部データシートは一番左のシートに対してBigQueryから連携するデータをセットしてください。(一番左のシート)<br>
